Mémory

Le jeu de mémory consiste à reconstituer des paires d'images. Le joueur retourne deux cartes. Si elles sont identiques, les cartes restent face visible, sinon elles reviennent face cachée. Le jeux se termine lorsque toutes les paires ont été trouvées. Vous pouvez paramétrer le nombre de paires pour votre partie.
